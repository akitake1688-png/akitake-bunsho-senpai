# 秋武文书哨兵 | Bunsho Senpai V1.0

专注志望理由书・研究计划书・小论文的东大基准审计工具。

## 功能
- 支持 PDF/Word/TXT 上传
- 逐步推理 + 实战案例诊断
- 温和建议 + 修改范例 + 伏笔埋设
- 东大基准 + DeepSeek-R1 reasoning 风格

## 快速使用
1. 访问在线版本：https://akitake1688-png.github.io/akitake-bunsho-senpai/
2. 点击左侧“上传文书”按钮上传您的文件
3. 深度逐段重构请加微信 qiuwu999

## 技术栈
- 前端：HTML + CSS + JavaScript
- PDF 解析：pdf.js
- DOCX 解析：mammoth.js
- 知识库：knowledge.json（RAG 风格）
